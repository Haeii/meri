



Merhaba! Ben 16 yaşında, yazılım dünyasına yeni adım atmış bir lise öğrencisiyim. Bu, benim GitHub üzerindeki ilk projem.

Proje Amacı
Bu proje, Python öğrenme sürecimde geliştirdiğim bir çalışmadır. Kendi hayatımdan alıntılar yaparken aynı zamanda yazılımın temel mantığını ve kütüphane kullanımını kavramayı hedefledim.

Yapay Zeka Desteği
Yazılım öğrenme sürecimde, kullanmam gereken kütüphaneler (library) ve bunların spesifik komutları hakkında henüz yeterli tecrübem olmadığı için yapay zekadan rehberlik alarak bu projeyi hayata geçirdim. Amacım, kodun çalışma mantığını anlamak ve kendimi bu alanda geliştirmektir.

Nasıl Çalıştırılır?
Projeyi bilgisayarınıza indirin (Download ZIP) ve klasörü dışarı çıkartın.

Bilgisayarınızda Python'ın yüklü olduğundan emin olun.

Proje klasörünün içinde bir terminal (komut satırı) açın.

Gerekli kütüphaneleri yüklemek için şu komutu çalıştırın:

Bash
pip install -r requirements.txt
Projeyi başlatmak için şu komutu kullanın:

Bash
python dosya_adi.py
(Not: dosya_adi.py kısmını projenin ana dosya ismiyle değiştirmeyi unutmayın.)

Projenin kodunda veya mantığında eksik/hatalı bulduğunuz yerleri belirtirseniz çok sevinirim. Kendimi geliştirmek ve ileride bu alanda profesyonelleşmek en büyük hedefim.
